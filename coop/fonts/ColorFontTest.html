Where can I use color fonts??

If you are trying to use a color font on Chrome and it's not working, it's because Chrome does NOT currently support OpenType-SVG fonts, but it does allow COLR based OpenType fonts. Also, you can try creating a layered font (one font file per color), and using some javascript from Manufactura Independente.

For fuller discussion of color fonts, see this great overview at colorfonts.wtf. That page has a great list of compatibility with various proprietary software, but NOT any open source software, so here goes, as of January 2021:
